## 项目介绍
该项目是一款影视介绍与查询的移动端项目，包含登录、信息搜索、影视信息、影人信息、评分、收藏、个人中心、头像上传等多个功能模块    

## 安装
```
npm install
```

## 开发环境启动
```
npm run serve
```

## 生产环境编译
```
npm run build
```
